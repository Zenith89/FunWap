
To Interperate
1)open the FUNWAP_INTERPERATOR Directory
2) open command prompt/ from the list of file in the directory click cmd

	FUNWAP_INTERPERET.exe script.fa
	Example=> FUNWAP_INTERPERET.exe  basics.fa
============================================================================================================

 To Compiling and Generating IL Code 
 
1)open the FUNWAP_COMPILER Directory
2) open command prompt or alternatively from the list of file in the directory click cmd
	FUNWAPCOMPILE.exe <script.fa>
	Example=> FUNWAPCOMPILE.exe  basics.fa
	 after the compeletion of the succesful compilation process you will find this mesage 
	 *********** DISASSEMBLY COMPLETE ***********************  press enter to continue 
	 now you will get outPut.exe and  outPut.txt in your working directory
